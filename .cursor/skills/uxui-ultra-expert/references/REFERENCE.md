# UXUI Ultra Expert Reference

## Checklist de revisão rápida

### Clareza

- O objetivo da tela está claro em 3 segundos?
- Existe uma ação principal evidente?
- Os rótulos estão específicos?

### Hierarquia

- O título principal realmente domina?
- O CTA principal é visualmente prioritário?
- Itens secundários estão competindo com o principal?

### Layout

- Há alinhamento consistente?
- O espaçamento segue ritmo visual?
- Existe respiro suficiente?

### Tipografia

- Os tamanhos têm contraste suficiente?
- O texto está confortável para leitura?
- Labels, meta info e helper texts estão legíveis?

### Cor

- O contraste é suficiente?
- A paleta tem função?
- Há excesso de cor chamativa?

### Componentes

- Estados existem e são consistentes?
- Hover, focus, disabled, error e loading foram pensados?

### Formulários

- Existe fricção desnecessária?
- Os erros ajudam o usuário a corrigir?
- O formulário parece confiável?

### Mobile

- O layout foi realmente adaptado?
- Os toques são confortáveis?
- A ordem da informação continua correta?

### Acessibilidade

- Foco visível
- contraste adequado
- labels corretos
- semântica razoável
- não depender só de cor

## Regras de ouro

- Clareza antes de estilo
- Hierarquia antes de detalhe
- Consistência antes de novidade
- Função antes de decoração
- Acessibilidade antes de perfeccionismo visual
- Sistema antes de improviso
